== Getting Started

1) Git clone or unzip the files into your directory

2) Browse the folder directory called "Mivly" with the following command in your console or terminal:

  cd mivly

3) Migrate database with the following command:

  rails rake:db
  
4) Start your localhost server with the following command:

  rails s
